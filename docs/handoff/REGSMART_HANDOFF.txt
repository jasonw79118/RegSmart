Project direction:
RegSmart is a Salesforce-style, cloud-based compliance operating system for banks and regulated financial institutions. It should support large complex institutions and scale down for small banks.

Current architectural direction:
- Cloud based, browser accessible, terminal agnostic
- Modular architecture for constant upgrades
- Frontend and backend separated
- Versioned APIs
- SSO-ready auth structure from the start
- Multi-tenant aware
- Role- and scope-based access model
- Audit tracker and issue tracker are first-class modules

Primary modules in scope:
1. Authentication and institution setup
2. Dashboard shell
3. Issue tracker
4. Audit/review tracker
5. Evidence and upload center
6. Administration

Key product goals:
- Track issues from identification through closure
- Import findings from internal audit, compliance, external auditor, examiner, and other outside sources
- Support privacy-aware uploads, masking, forgetting, and retention controls later
- Prepare for AI-assisted review, extraction, and scoring
- Be suitable for U.S. financial institution usage
