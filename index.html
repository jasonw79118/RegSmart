Help / User Guide

  • Login: use the sign-in screen to access the shell and confirm live deployment.
  • Dashboard: shows issue pressure, review activity, and enterprise oversight.
  • Issues register: click any issue row to open the slide-out detail panel and work the issue through the lifecycle.
  • Issue lifecycle: examiners expect a clear trail from identification through ownership, evidence, validation, and closure.
  • Example: a BSA alert escalation issue should stay open until remediation evidence and validation results are both retained.
  • Version control: the active build version remains visible in the bottom-left.
